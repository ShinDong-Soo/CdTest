

## ✅ \[C# 기초] 두 수의 나눗셈 후 1000 곱하기 문제 풀이

---

### 🔍 문제 요약

* 두 정수 `num1`, `num2`가 주어진다.
* `num1 / num2`를 계산 → `결과에 1000을 곱함` → `정수 부분만 반환`
* **실수 연산이 필요한 문제!**

---

### 🎯 문제를 푸는 사고 과정

| 단계 | 생각할 질문            | 해석                                           |
| -- | ----------------- | -------------------------------------------- |
| 1  | 입력은 무엇인가?         | 정수 `num1`, `num2`                            |
| 2  | 무슨 연산을 해야 하는가?    | `num1 ÷ num2` 후 `×1000`                      |
| 3  | 정수 나눗셈 vs 실수 나눗셈? | 소수점까지 필요 → **실수형 나눗셈**                       |
| 4  | 자료형은 어떻게 정해야 하나?  | `double` 또는 `float`                          |
| 5  | 최종 반환은?           | 정수형으로 **소수점 버림**해야 함 → `Math.Floor` or `형변환` |

---

### 💡 핵심 문법 요약

* `double result = (double)num1 / num2;`
  → 정수끼리 나누면 **소수점 무시**되므로 `형변환`이 필요함
* `int intResult = (int)(result * 1000);`
  → 1000 곱한 뒤 정수형으로 변환

---

### ✅ 최종 코드

```csharp
public class Solution {
    public int solution(int num1, int num2) {
        double result = (double)num1 / num2;  // 실수형 나눗셈
        return (int)(result * 1000);          // 1000 곱한 후 정수로 변환
    }
}
```

---

### 🧪 입출력 예시 분석

| num1 | num2 | 나눗셈    | 곱하기 1000 | 정수 결과 |
| ---- | ---- | ------ | -------- | ----- |
| 3    | 2    | 1.5    | 1500.0   | 1500  |
| 7    | 3    | 2.3333 | 2333.333 | 2333  |
| 1    | 16   | 0.0625 | 62.5     | 62    |

---

### 📌 Notion용 요약 정리

* ✅ **정수끼리의 나눗셈은 몫만 남는다** → **실수 연산 위해 형변환 필요**
* ✅ `int / int` → `int`
  `double / int` 또는 `(double)int / int` → `double`
* ✅ `double * 1000` → 소수점 존재
  → `(int)` 또는 `Math.Floor()`로 정수 부분만 취함
* ✅ 이 문제는 **실수 계산 후 정수 변환**을 연습할 수 있는 좋은 예시
